  # Description du jeu
  
Un joueur est dans une bibliothèque universitaire et a pour objectif de retrouver un livre rare avant la fermeture de la bibliothèque.
Pour l'aider, il pourra résoudre des enigmes qui lui donneront des indications sur la direction à prendre ainsi que des clès permettant d'accéder à certaines salles.
